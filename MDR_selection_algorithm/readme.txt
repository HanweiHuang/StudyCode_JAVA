identify all the MDRs and their connectivity (adjacency) with 
other MDRs and non-MDRs using the MDR selection algorithm

There are three java files, node.java, findmdr.java, findmdr_xls.java

two versions for this project, findmdr.java will just output the result on the screen, and findmdr_xls.java will create .xls file. 
You should import jxl-2.6.10.jar into your eclipse if you run findmdr_xls.java
